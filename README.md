# Gym Management System (Flask)

## Project Description
This project is a basic Gym Management System website developed using Flask, HTML, and CSS.
It demonstrates how Flask routes are used to render multiple web pages using templates and static files.

## Pages Included
- **Home Page**: Displays a welcome message and gym highlights
- **About Us Page**: Shares information about the gym’s mission, team, and facilities
- **Membership Page**: Shows different membership plans and services

## Technologies Used
- Python (Flask)
- HTML
- CSS
